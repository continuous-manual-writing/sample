# ドキュメント作成システム構築ガイド Appendix 1 サンプル

本ディレクトリには本章のサンプルファイルが収録されています。以下、サンプルファイルについて解説します。

## HTMLドキュメントの生成

AsciiDocファイルからHTMLフォーマットのドキュメントを生成します。生成されるドキュメントは2通り（スタイル適用なし・あり）です。スタイル適用による外観の変化を確認できます。

### 実行方法

生成には以下のコマンドを実行します。

```
$make html
````

## 必要なソフトウェア

以下の環境での実行を確認しています。

- Asciidoctor v1.5.2
- Ruby v2.1.7p400
